# weather

## API keys
For api calls you need to replace the value of the key "VUE_APP_OWM_KEY" in the ".env" file with a real key that can be obtained [here](https://home.openweathermap.org/api_keys).  

For example:
```
VUE_APP_OWM_KEY=*YOUR_API_KEY*
```
